 - 滚动计数窗口
 - 滑动计数窗口



滚动窗口:
Tumbling Windows 将数据依据固定的窗口长度对数据切分,时间对齐,长度固定,没有重叠 ,[ ) 左闭右开
例:
[       )
        [       )
                [       )
                        [       )
滑动窗口:
Sliding Windows
滑动窗口是固定窗口的更广义的形式, 滑动窗口由固定的窗口长度 和 滑动间隔 组成
窗口固定, 可以有重叠
例:
[       )
    [       )
        [       )
            [       )


会话窗口:
由一系列事件组合一个指定时间长度的timeout间隙组成,也就是一段时间没有接收到新数据就会生成新的窗口
特点: 时间无对齐
例:
w1 [       ) w2 [       ) w3  -----> 间隙称为 session gap
间隙固定,时间不对齐
